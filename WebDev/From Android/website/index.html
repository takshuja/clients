What is Programming?

Programming is a way to talk to the computer or if we wanna be more precise, it means to Instruct the computer to perform a specific task.

Let's say you need a computer to check if a number is less than the other. We will tell the computer the following instructions:
    Ask the user to enter a number store it in a variable.
    Ask the user to enter another number and also store it in another variable.
    Compare the two numbers.
    Tell the user the end result.

In the above statement, we told the computer to perform a specific instruction for us or in other words, we "programmed" the computer to perform a specific task. But, here comes our first problem. Niether we understand the language of computers that is 0 and 1 nor the computer knows anything about human languages. So, in order to overcome this problem, we created some special languages to communicate with the computers known as "programming languages".

Some of the commonly used programming languages are: Python, C, C++, Java, Kotlin, Golang, Rust, Javascript, C# and many more.

Types of programming languages:

1. Low Level Languages: The programming languages that are very close to writing actual machine instructions. These are often known as computers native languages and they are machine dependent languages.
Some examples of low level languages are: ASM and machine code.

2. Mid Level Languages: These programming languages are close to machine as well as human readable. They are much easier to learn than low level programming languages.
Some examples of mid level programming languages are: C, C++, Java.

3. High Level Languages: These programming languages are much closer(more human readable) to humans than machines. These languages are the easiest to learn and debug.
Some examples of high level languages are: Python, Perl, Ruby, C#, ...



Now, lets dive into some basic concepts of programming.

1. Variables: Variables are one of the basic and most important building blocks of any program. Variables are names which we give to computer memory locations to store data. We can think of variables as containers to store some form of data. When we declare a variable, we tell the computer to store a value into a computer memory address so we can access it later.
